<-PREVIEW_CREA->

Preview_crea est un document HTML simpliste qui se charge d'afficher toutes vos creas afin de les visualiser rapidement au meme endroit.

Tout les formats sont presents il suffit juste de retirer le commentaire sur le format desiré et de remplir src avec le chemin vers votre index.html.

Une fois que tout est bon ouvrir le fichier HTML dans un navigateur.

Tout les formats dervrait aparraitre a la bonne taille... si dans une iframe un scroll est proposé, il suffit de modifier la width et la height afin de l'agrandir.
